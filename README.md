# EnOceanRubySample
サンプルプログラムです。

## Description
Raspberry Pi 2 Model B などに EnOcean受信モジュールである USB400J を接続し、  
EnOceanのシリアル通信で取得したデータをクラウドにアップします。   


## Requirement
*Linux  
*Ruby  
*Ruby-SerialPort  
*curl  


## Usage
$ ruby sample.rb
